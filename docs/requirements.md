## 退休计算器 — 需求文档

### MVP 目标
上线一个微信小程序，用户可以计算中国社保养老金、规划退休储蓄、查询延迟退休年龄，并将结果通过转发或生成图片分享。

### 用户故事

1. **延迟退休查询**：小王是90后女职工，想知道自己到底多少岁能退休，输入出生日期后马上看到结果。
2. **社保养老金计算**：老张在北京工作25年，想算算退休后每月能领多少养老金，选择城市填几个参数就知道了。
3. **退休储蓄规划**：小李30岁开始存钱，想规划退休生活，输入当前储蓄和每月存款后看到预估的每月可支配金额。
4. **分享结果**：算完之后可以转发给朋友，或者生成一张图片保存到相册。

### 页面清单
| 页面 | 功能 |
|------|------|
| 首页 | 三个计算器入口按钮，简洁引导 |
| 退休年龄查询 | 表单 + 结果展示 |
| 社保养老金 | 城市选择 + 表单 + 结果展示 |
| 退休储蓄 | 表单 + 结果图表 |
| 结果页 | 通用结果卡片 + 分享按钮 |
